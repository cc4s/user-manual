1 CC4S User Documentation

Welcome to the user documentation of the cc4s code.

1.1 Getting started

The Quickstart section explains everything you need to know about downloading the code, compiling it and running a few tests.

1.2 How to control input for CC4S?

This Input section describes how YAML files are used to control CC4S calculations.

1.3 Algorithms in CC4S

The Algorithms section summarizes and explains the most important algorithms available in CC4S.

1.4 Interfaces to CC4S

This section describes the available interfaces between electronic structure theory codes and CC4S. Interfaces to the following codes are currently available

  • The interface to the Vienna ab initio simulation package is described in the VASP section.
  • The interface to the FHI-AIMS code is currently under work but will be described in the FHI-AIMS section soon.

1.5 Tutorials and examples

To learn how to run CC4S calculations and interpret its output, we provide the following tutorials

  • LiH solid using CCSD(T)
  • Water adsorption energy on a LiH surface on the level of CCSD(T)
  • Energy difference between r-BN and c-BN crystal phases

In order to get started using cc4s you can start reading the quick start tutorial.
